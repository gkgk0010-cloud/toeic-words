# 404 나올 때 — 체크리스트

---

## 1. GitHub에 api 폴더가 있는지 확인

1. **https://github.com/gkgk0010-cloud/toeic-words** (또는 본인 toeic-words 저장소) 접속
2. 파일 목록에 **api** 라는 **폴더**가 보이는지 확인
3. **api** 폴더를 클릭했을 때 안에 **notion-words.js** 가 있는지 확인

**없으면** → 404 원인. **api 폴더 전체**를 GitHub에 올려야 함.

**올리는 방법 (간단):**
1. GitHub 저장소 페이지에서 **Add file** → **Upload files**
2. 로컬 **toeic-words** 폴더에서 **api** 폴더 **통째로** 드래그해서 올리기 (또는 api 폴더 안에 들어가서 notion-words.js 선택해서 올리기)
3. **Commit changes** 클릭
4. Vercel이 자동 배포하면 1~2분 뒤에 **다시** 아래 주소 열어보기

---

## 2. Vercel Root Directory

- 저장소가 **toeic-words만** 있으면 → Root Directory **비우기** (빈 칸)
- Vercel → toeic-words-b5vc → Settings → Build and Deployment → Root Directory 확인 후 **Save** → **Redeploy**

---
