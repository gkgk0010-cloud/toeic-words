# 연결사 전용 페이지 (connector.html) — 내일 개강용

`?db=...` 파싱·캐시 문제 없이 **연결사만** 열리게 했습니다.

---

## 어떻게 동작하는지

| 주소 | 내용 |
|------|------|
| `toeic-words-b5vc.vercel.app/` | 시제·부사 (words.json) |
| `toeic-words-b5vc.vercel.app/connector.html` | **연결사만** (노션 API, 고정 DB ID) |

- **connector.html** 은 **항상** 연결사 DB만 불러옵니다. URL에 `?db=` 없어도 됩니다.
- 똑패스에서 "연결사(접속부사) 테스트" 버튼을 누르면 **connector.html** 로 열립니다.

---

## 배포 후 할 일 (딱 2가지)

1. **toeic-words** 수정본을 **Git에 푸시** (connector.html, app.js, gas_src/index.html 포함)
2. **clasp push** 한 번 (똑패스 앱에 연결사 → connector.html 반영)

---

## 확인 방법

- **직접 주소로:**  
  `https://toeic-words-b5vc.vercel.app/connector.html`  
  → 연결사 카드/퀴즈가 나오면 성공.
- **똑패스에서:** 시험 만점 족보 → **연결사(접속부사) 테스트** → 연결사가 나오면 성공.
