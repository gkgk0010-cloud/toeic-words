# toeic-words-b5vc — "다른 게 배포되고 있는지" 확인

---

## 1. Vercel에서 연결된 저장소 확인

1. **Vercel** → **toeic-words-b5vc** 프로젝트 클릭
2. **Settings** → **Git** (또는 **Build and Deployment**)
3. **Connected Git Repository** 에 나오는 저장소 이름 확인  
   - 예: `gkgk0010-cloud/toeic-words` 인지  
   - 아니면 `gkgk0010-cloud/똑패스` 같은 **다른 저장소**인지

---

## 2. Root Directory 확인

같은 Settings 화면에서 **Root Directory** 확인.

- **비어 있음**  
  → 배포되는 건 **저장소 루트** 안의 파일들.  
  → `connector.html` 은 **저장소 루트**에 있어야 함 (예: repo 루트에 index.html, connector.html, api/ 가 같이 있음).

- **`toeic-words`** 로 되어 있음  
  → 배포되는 건 **그 저장소 안의 `toeic-words` 폴더** 안의 파일들만.  
  → `connector.html` 은 반드시 **그 저장소의 `toeic-words` 폴더 안**에 있어야 함.  
  → repo 루트에 connector.html 을 올리면 배포에 **포함되지 않음** (404 원인).

---

## 3. GitHub에서 connector.html 위치 확인

**연결된 그 저장소**를 GitHub에서 연다.

- **Root Directory가 비어 있는 경우**  
  - 저장소 루트에 `index.html`, `connector.html`, `api/` 등이 보여야 함.  
  - `connector.html` 이 **루트**에 없으면 → 루트에 올리거나, 루트에 있는 폴더를 열어서 그 안에 넣지 말고 **최상위**에 두어야 함.

- **Root Directory가 `toeic-words` 인 경우**  
  - 저장소에 **toeic-words** 라는 폴더가 있어야 함.  
  - **그 폴더 안**을 연다.  
  - 그 안에 `index.html`, `connector.html`, `api/` 가 같이 있어야 함.  
  - `connector.html` 이 **toeic-words 폴더 밖**(다른 폴더나 루트)에만 있으면 → **toeic-words 폴더 안으로** 옮겨야 함.

---

## 4. 정리

| Root Directory (Vercel) | connector.html 있어야 하는 위치 (GitHub) |
|--------------------------|------------------------------------------|
| 비어 있음                | **저장소 루트** (index.html 과 같은 단계) |
| `toeic-words`           | **toeic-words 폴더 안** (index.html 과 같은 단계) |

"다른 게 배포되고 있다" = 연결된 저장소 또는 Root Directory 때문에 **우리가 수정한 폴더가 아닌 다른 경로**가 배포되고 있을 수 있음.  
위처럼 **연결된 저장소**와 **Root Directory**, **connector.html 위치**만 맞추면 됨.
