# toeic-words 테스트 방법

시제·연결사·인칭대명사가 각각 자기 내용만 나오는지 확인하는 방법입니다.

---

## 1. 시제·부사 테스트

**열기:** 아래 주소를 그대로 주소창에 넣고 엔터

```
https://toeic-words-b5vc.vercel.app/
```

**확인할 것**
- 탭 제목: `토익 시제부사 · 똑패스`
- 페이지 제목: `토익 시제부사`
- 필터 라벨: `시제`
- 필터 옵션: 전체 / 현재 / 과거 / 미래 / 현재완료
- 카드·퀴즈: 시제부사 단어만 (현재/과거/미래 등)

---

## 2. 연결사(접속부사) 테스트

**열기:** (연결사 노션 DB ID 사용)

```
https://toeic-words-b5vc.vercel.app/?db=2fa6e4c35a0e81cda20ac619508bbeea
```

**확인할 것**
- 탭 제목: `연결사(접속부사)` 또는 노션 DB 제목 + `· 똑패스`
- 필터 라벨: `카테고리` (또는 노션 속성 이름)
- 카드·퀴즈: 연결사 단어만

---

## 3. 인칭대명사 테스트

**열기:** (인칭대명사 노션 DB ID 사용)

```
https://toeic-words-b5vc.vercel.app/?db=3016e4c35a0e807ea96af840fc6f6a6a
```

**확인할 것**
- 탭 제목: `인칭대명사표` 또는 노션 DB 제목 + `· 똑패스`
- 필터 라벨: `구분` 또는 노션에 넣은 그 컬럼 이름
- 필터 옵션: 단수, 복수, 1인칭 등 실제 DB에 있는 값만
- 카드·퀴즈: 인칭대명사만 (시제/연결사 단어 없음)

---

## 4. 똑패스 앱에서 열어보기

1. 똑패스(GAS 웹앱) 로그인
2. **테스트 진행하기** → **시제·부사 / 연결사 / 인칭대명사** 중 하나 선택
3. 새 탭으로 단어 앱이 열리면, 위 1~3번처럼 **제목·필터·내용**이 그 테스트에 맞는지 확인

---

## 5. 섞임 방지 확인 (인칭대명사)

1. 먼저 **시제·부사** URL로 한 번 연 다음 닫기
2. 그다음 **인칭대명사** URL로 열기  
   → 탭 제목이 "인칭대명사표", 필터가 "구분", 카드가 인칭대명사만 나와야 함 (시제부사 안 섞임)
3. **Ctrl+Shift+R**(강력 새로고침) 후에도 같은지 확인

---

## 6. 연결사·인칭대명사가 바로 나오게 (앱처럼)

- **연결사**: `data/connector-words.json`을 먼저 불러와서 **첫 방문부터** 바로 표시됩니다.
- **인칭대명사**: 한 번이라도 연 적 있으면 **localStorage**에 저장돼서, 다음에 열 때(다른 탭·다른 날 포함) **바로** 표시됩니다.  
  **첫 방문부터** 바로 나오게 하려면, 터미널에서 한 번만 실행해서 `data/pronoun-words.json`을 채운 뒤 커밋·배포하면 됩니다.
  ```bash
  cd toeic-words
  NOTION_API_KEY=secret_xxxx node scripts/fetch-pronoun-words.js
  ```
  (`NOTION_API_KEY`는 노션 연동봇 비밀번호. 실행 후 `data/pronoun-words.json`에 단어가 채워지면 그 파일을 커밋하고 Vercel 배포.)
- 각 페이지는 **캐시·정적 JSON 모두 해당 db만** 사용하므로 시제/연결사/인칭대명사 내용이 섞이지 않습니다.

---

## DB ID가 다를 때

똑패스 `index.html` 안 `WORDS_APP_TESTS` 배열에 들어 있는 `db` 값이 실제 노션 DB ID와 같아야 합니다.  
연결사/인칭대명사 DB ID를 바꿨다면 그곳도 같은 ID로 수정한 뒤 **clasp push** 하면 됩니다.
