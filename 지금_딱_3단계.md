# 지금 딱 3단계

## 1. GitHub에서 확인

- 저장소 열기
- **toeic-words** 라는 **폴더**가 보이나요?
- 그 폴더 안을 열면 **index.html, connector.html, app.js** 가 보이나요?

→ **예** → 2번으로  
→ **아니오** (파일이 루트에만 있음) → 로컬에서 아래처럼 push 한 번 하면 폴더 구조로 올라감. 그다음 2번.

---

## 2. 로컬에서 push 한 번

1. 똑패스 폴더에서 터미널 열기
2. `git add .`
3. `git commit -m "toeic-words 구조 맞춤"`
4. `git push`

끝. (Vercel은 push 하면 자동 배포)

---

## 3. 똑패스 앱(GAS) 반영

- `clasp push` 한 번 (테스트 링크 주소 반영)

---

**정리:**  
2번 push → 3번 clasp push → 끝.  
나머지(api, WORDS_APP_URL)는 이미 코드에 넣어 둔 상태라 건드릴 거 없음.
