# 단어 앱(toeic-words) — Vercel 배포 방법

정적 파일(HTML/CSS/JS)이라 **빌드 없이** 올리면 됩니다.

---

## 방법 A: Vercel 웹에서 배포 (추천)

### 1. 저장소 준비
- **toeic-words 폴더만** 배포하려면:
  - **옵션 1:** GitHub에 **똑패스 전체** 저장소가 있으면, Vercel에서 프로젝트 불러올 때 **Root Directory**를 `toeic-words`로 지정.
  - **옵션 2:** **toeic-words만** 새 GitHub 저장소로 만들기.
    - GitHub에서 새 저장소 생성 (예: `toeic-words`) — **"Add a README" 등 체크하지 말고 빈 저장소로 생성**
    - 로컬에서 **반드시** 아래처럼 코드를 푸시한 뒤에 Vercel에서 Import 해야 함.
    - **GitHub 사용자명이 `gkgk0010-cloud`일 때** (다르면 URL만 본인 저장소 주소로 바꾸기):
      ```bash
      cd toeic-words
      git init
      git add .
      git commit -m "init: toeic-words app"
      git branch -M main
      git remote add origin https://github.com/gkgk0010-cloud/toeic-words.git
      git push -u origin main
      ```
    - 푸시가 끝난 후에 Vercel에서 **Add New → Project** → 해당 저장소 선택 → Deploy.

**⚠ "The provided GitHub repository does not contain the requested branch or commit reference" 오류가 나올 때**  
→ GitHub 저장소가 **비어 있는 상태**입니다. 위 `git init` ~ `git push` 단계를 **먼저** 실행해서 코드를 올린 다음, Vercel에서 다시 Deploy 하세요.

### 2. Vercel에서 배포
1. https://vercel.com 접속 → **Login** (GitHub로 로그인)
2. **Add New… → Project**
3. **Import** 할 저장소 선택
   - **똑패스 전체** 저장소를 쓴 경우: **Root Directory**에 `toeic-words` 입력 후 **Continue**
   - **toeic-words만** 있는 저장소: Root Directory 비워 두고 **Continue**
4. **Framework Preset:** "Other" 선택 (또는 그대로 두고)
5. **Build Command:** 비워 두기 (또는 비워도 됨)
6. **Deploy** 클릭

### 3. 배포 완료 후
- **Visit** 클릭하면 단어 앱 URL이 나옵니다.
- 예: `https://toeic-words.vercel.app` (프로젝트 이름에 따라 다름)
- 이 URL을 **gas_src/index.html** 의 `WORDS_APP_URL` 에 넣으면 됩니다.

## 방법 B: Vercel CLI로 배포 (Git 없이)

1. Vercel CLI 설치 (한 번만):
   ```bash
   npm i -g vercel
   ```
2. toeic-words 폴더에서:
   ```bash
   cd "c:\Users\admin\OneDrive\바탕 화면\똑패스\toeic-words"
   vercel
   ```
3. 로그인 안내 따라 로그인 (브라우저 열림)
4. 질문에:
   - **Set up and deploy?** → Y
   - **Which scope?** → 본인 계정 선택
   - **Link to existing project?** → N
   - **Project name?** → toeic-words (또는 원하는 이름)
   - **In which directory is your code?** → ./ (엔터)
5. 끝나면 **Production URL** 이 나옵니다. 이걸 `WORDS_APP_URL` 에 넣으면 됩니다.
